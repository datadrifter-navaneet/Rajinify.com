# Rajinify.com

Rajinify - AI-Powered Rajinikanth Transformation App
===================================================

Rajinify is a web application that transforms any uploaded photo into Rajinikanth style using AI image processing. Built for fans and developers, it features real-time face detection, style transfer, and one-click downloads.

Features
--------
- Upload any photo and get Rajinikanth-fied version instantly
- Real-time preview with AI style transfer
- Multiple Rajini styles (Superstar, Young Rajini, Action Hero)
- One-click download in HD quality
- Responsive design for mobile and desktop

Demo
----
Live Demo: https://yourusername.github.io/rajinify/ (Deploy via GitHub Pages)

Tech Stack
----------
- Frontend: HTML5, CSS3, Vanilla JavaScript
- AI/ML: TensorFlow.js for browser-based face detection and style transfer
- Image Processing: Canvas API for real-time transformations
- No backend required - fully client-side

Getting Started
---------------
1. Clone the repo: git clone https://github.com/yourusername/rajinify.git
2. Open index.html in any modern browser
3. Upload photo and watch the magic!

Usage
-----
- Click "Upload Photo" and select image
- Choose Rajinikanth style from dropdown
- Preview transformation live
- Download final image with "Save Rajinified"
